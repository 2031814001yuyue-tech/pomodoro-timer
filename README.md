# 🍅 番茄钟

一个简洁精美的番茄工作法计时器，支持多用户、自定义主题、背景白噪音。

## 功能

- **⏱ 番茄工作法** — 专注 25 分钟 → 短休息 5 分钟 → 长休息 15 分钟循环
- **👤 多用户** — 每人独立记录和配色方案
- **🎨 自定义主题** — 6 套预设主题 + 自由调色
- **🎵 背景音** — 内置白噪音、雨声、海浪、森林，程序化生成无需外部文件
- **📋 专注记录** — 自动记录每次专注，支持导出/导入备份
- **⌨️ 快捷键** — Space（暂停/继续）、R（重置）、S（跳过）

## 使用

直接在浏览器打开 `index.html` 即可，无需服务器、无需安装。

## 部署

支持 GitHub Pages：仓库 Settings → Pages → 选择 main 分支 → 保存，即可获得在线链接。

## 技术栈

纯 HTML + CSS + JavaScript，零依赖，所有声音通过 Web Audio API 实时生成。
